# Features

Inkstone focuses on static documentation generation, not dynamic application features.

The feature set is centered on:

- Markdown rendering
- theme rendering
- syntax highlighting
- demo blocks
- client-side search
- static deployment output

The generated site does not require a server, database, queue, authentication, or external search provider.
